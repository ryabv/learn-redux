# Learn Redux

Запуск проекта:<br />
1. ```npm i```;<br />
2. ```npm run server``` или ```gulp server```.

К основным файлам проекта относятся:

- ```index.html```
- ```style.css```
- ```img/*```
- ```js/script.js```
- ```js/data.js```

Основная реализация находится в файле ```script.js```, файл ```data.js``` необходим для начального стейта класса ```Store```.
